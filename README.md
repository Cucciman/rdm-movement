# RdM — Redistribution Movement

**Capitalism grows the pie. We are here to make sure it is shared.**

Website: [rdm-movement.org](https://rdm-movement.org)

RdM is a pan-European civic movement advocating for a European fiscal 
union to fund universal free healthcare and education for every EU citizen.
